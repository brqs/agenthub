# Agent Logo Assets

Local provider logo assets used by `AgentAvatar`.

- `openai.svg`: Simple Icons OpenAI SVG.
- `anthropic.svg`: Simple Icons Anthropic SVG, used for `provider: "claude"`.
- `deepseek.svg`: Wikimedia Commons DeepSeek icon SVG.
If the team later receives an official brand asset pack, replace these files in place and keep the filenames stable so UI code does not need to change.

Local role agents such as Orchestrator and Web Designer use `lucide-react` icons in `AgentAvatar` instead of stored SVG logo files.
